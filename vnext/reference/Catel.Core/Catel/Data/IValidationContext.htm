IValidationContext

Name Value
Assembly Catel.Core
Namespace Catel.Data
Available on .NET Framework 4.5, .NET Framework 4.6, Portable Class Libraries, Windows 10.0 (Universal Apps), Xamarin - Android, Xamarin - iOS 
public interface IValidationContext

Interface describing the validation

Properties

HasErrors

Gets a value indicating whether this instance contains errors.

HasWarnings

Gets a value indicating whether this instance contains warnings.

LastModified

Gets the last modified date/time.

Note that this is just an informational value and should not be used for comparisons. The [DateTime](#) 
is not accurate enough. Use the ```LastModifiedTicks``` instead.

LastModifiedTicks

Gets the last modified ticks which is much more precise that the LastModified. Use this value to compare last modification ticks on other validation contexts.

Because only full .NET provides a stopwatch, this property is only available in full .NET. All other target frameworks
will return the Ticks which is ```not``` reliable.
